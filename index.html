  • Passe o mouse por cima dos ícones para ler a descrição.

  • HTML é uma linguagem de marcação que permite definir a estrutura básica do site. É através dele, por exemplo, que inserimos textos, imagens e conteúdos.

  • CSS é uma linguagem de estilização. Por meio dele que se faz a personalização e padronização do site, de acordo com o design desejado.

  • Javascript é uma linguagem de programação de alto nível, voltada para o desenvolvimento web. Ela permite a criação de sistemas complexos e possibilita uma maior interatividade com o usuário.

  • C++ é uma linguagem de programação orientada a objeto que também permite a criação de sistemas complexos.

  • O Figma é um software de criação gráfica voltado para o desenvolvimento de sites e aplicativos. Através dele é possível visualizar como ficaria o site final, em relação ao design.
